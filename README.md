# NameChanger

A Minecraft mod for Forge and Fabric that lets you change player display names without modifying their actual GameProfile.

## Features

- Set custom nicknames with `/nick name <nickname>` or `/nick <nickname>`
- Reset nicknames with `/nick reset`
- Hide all name tags with `/nick hide on/off` (requires permission level 2)
- Hold **U** key to reveal real names when hiding
- Nicknames appear on:
  - Overhead name tags
  - Tab list
  - Chat messages
- Plasmo Voice compatibility

## Supported Versions

| Version | Platform | Status |
|---------|----------|--------|
| 1.20.1  | Forge, Fabric | ✅ Active |
| 1.21.1  | NeoForge, Fabric | ✅ Active |

## Installation

1. Install the required mod loader (Forge/Fabric/NeoForge)
2. Install [Architectury API](https://modrinth.com/mod/architectury-api)
3. Download the mod jar and place it in your `mods` folder
